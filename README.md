## Patient No-Show Analysis Dashboard
## Project Overview


This project presents a Power BI dashboard analysing patient appointment attendance to identify patterns and factors influencing missed appointments.

## Objectives
•	Analyse patient attendance trends 


•	Identify factors contributing to no-shows 


•	Provide insights to improve healthcare service delivery 



## Tools Used
•	Power BI 


•	DAX 


•	Data Analysis 



## Dashboard Features


## Page 1: Executive Dashboard


•	Total Patients 


•	Show vs No-Show Analysis 


•	No-Show Rate (%) 


•	Gender Distribution 



## Page 2: Advanced Analysis


•	Age Group Analysis 


•	SMS Reminder Impact 


•	Socioeconomic Factor (Scholarship) 



## Key Insights


•	Approximately 20.3% of patients missed their appointments 


•	Patients aged 30–60 recorded the highest no-show rates 


•	SMS reminders showed limited impact on attendance 


•	Patients without scholarships had significantly higher no-show rates 



## Conclusion

 
The analysis highlights that socioeconomic factors and patient circumstances play a key role in healthcare attendance.
